# PhonePe UPI Payment App

A full-stack fintech application that simulates UPI-based payments similar to PhonePe.

## Tech Stack
Frontend: React.js  
Backend: Spring Boot  
Database: MySQL  

## Features
- User Registration
- Login using UPI ID & PIN
- Wallet Balance
- Send Money
- Add Money
- Transaction History
